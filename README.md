# 📈 Stockhouse 全能網頁小幫手 

專為 [Stockhouse](https://www.stockhouse.com.tw/) 設計的 Tampermonkey 瀏覽器擴充腳本，旨在優化使用的效率，讓操作更順暢、資料收集更方便。

---

### 🔔 非阻塞系統通知
受夠了每次系統跳出 `重複委託` 警告視窗時，都會打斷你的操作流程嗎？
* 本腳本將網站原生的 `alert` 徹底攔截，替換為現代化的**浮動式通知（Toast）**。
* 訊息會平順地出現在畫面上方，幾秒後自動消失，完全不干擾你的滑鼠與鍵盤操作。

### 📋 歷史通知紀錄一鍵複製
閃過去的通知來不及看？需要將系統訊息保留下來分析？
* 腳本會在背景自動記錄每一次的系統通知，並貼心地幫你加上 **[HH:MM:SS] 時間戳記**。
* 當系統產生第一筆通知時，網頁**左下角**會自動浮現一個「📋 複製通知紀錄」按鈕。
* 點擊按鈕，即可將所有歷史訊息一鍵複製到剪貼簿，方便轉貼或存檔。

### 📄 頁面擴增 1000 筆顯示上限選項
* 覺得每頁顯示的資料筆數太少？腳本會在下拉選單中自動補上 **「1000」** 的選項，讓你一眼看盡大量數據。

### 📂 已委託代領查詢頁面「展開全部」功能 
在檢視已委託代領查詢頁面時，不需要再一筆一筆手動點擊展開了。
* 腳本會在原有的「下載成EXCEL檔」按鈕右側，自動新增一顆 **「展開全部」** 按鈕。
* 點擊後即可瞬間展開當前頁面的所有隱藏資料列。


## 🛠️ 安裝指南

1. **擴充功能：** 首先，您需要在瀏覽器安裝使用者腳本管理器 [Tampermonkey](https://www.tampermonkey.net/)。
2. **安裝腳本：** 前往 [openuserjs](https://openuserjs.org/scripts/ssarcandy/Stockhouse_%E5%85%A8%E8%83%BD%E5%B0%8F%E5%B9%AB%E6%89%8B) 點擊頁面的 `Install` (安裝) 按鈕。
3. **確認啟用：** 安裝完成後，開啟或重新整理 Stockhouse 網頁，Tampermonkey 圖示上顯示數字即代表腳本已成功載入！
